catcher\local_node\下执行
只首次执行：
npm install

编译：
npm run build-ts

修改catcher\local_node\config.js文件中的
AppOptions
dir:指到游戏目录（该目录下有index.js）

catcher\local_node\dist下执行
node .\server.js
